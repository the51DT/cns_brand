Title

  • title:

    • text : 텍스트
    • subTitle : 타이틀 상단 텍스트
    • subDesc : 타이틀 하단 텍스트(subTitle과 동일한 스타일)
    • size : font-size (typograph.scss 참고)
    • class : font-weight, color (스타일 가이드, tailwind-all.scss 참고)

  • marginVertical: margin-top/bottom 간격에 대한 이중클래스 적용(spacing.scss, tailwind-all.scss 참고)
