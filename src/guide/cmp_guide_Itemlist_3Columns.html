Itemlist_3Columns

  • icon : icon 클래스

  • ImageAlign : 이미지 정렬

  • title : 타이틀 텍스트

  • titleSize : 타이틀 사이즈에 대한 스타일 이중 클래스 (font-size // 값이 없다면 .font-body-lg 적용)

  • titleStyle : 타이틀에 대한 스타일 이중 클래스 (font-weight, color 등)

  • content : 배열

  • textAlign : 텍스트 정렬(left, right, center)

  • itemType : type-box, type-dash, type-border-bottom, type-border-left

  • marginVertical: margin-top/bottom 간격에 대한 이중클래스 적용(spacing.scss, tailwind-all.scss 참고)
