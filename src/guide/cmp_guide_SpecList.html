SpecList

  • subTitle : text

  • title : text

  • content

    • size : title의 폰트 사이즈 (typograph.scss 참고)

    • color : title의 폰트 컬러 (스타일 가이드, tailwind-all.scss 참고)

    • addClass : title의 font-weight 등 클래스 추가(스타일 가이드, tailwind-all.scss 참고)

    • text : text (배열 : 1개 초과 시 bullet 타입으로 적용)

  • borderTop : 'used' (border-top 사용 여부), 'only-pc (pc에서만 노출)'

  • firstItemPadding: 'padding-top-0' / 'pc-padding-top-0' / 'mo-padding-top-0'
