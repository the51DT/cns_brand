ImageItem

  • typeClass : layout 적용

  • imageUrl : 이미지 url

  • imageRound : true 일 때 border-radius 적용 (기본 [32 / 20], type-left,right [32 / 16])

  • title : text (기본 스타일 클래스 적용 : .font-head-sm.font-medium.color-text-default)

    title 커스텀이 필요한 경우

    • { text : text

    • size : font-size (typograph.scss 참고)

    • class : font-weight 외 등등 (tailwind-all.scss, color.scss) 사용 시 컬러/font-weight 지정 필요}

  • content : text
