ImageItem

  • typeClass : layout 적용

  • imageUrl : 이미지 url

  • ImageUrlMo : 모바일 이미지 url - 사용 시 pc 이미지에 이중클래스 추가되어 분기에 따라 노출된다. (.img-box__only-pc / .img-box__only-mo)

  • imageRound : true 일 때 border-radius 적용 (기본 [32 / 20], type-left,right [32 / 16])

  • title : text (기본 스타일 클래스 적용 : .font-head-sm.font-medium.color-text-default)

    title 커스텀이 필요한 경우

    • { text : text

    • size : font-size (typograph.scss 참고)

    • class : font-weight 외 등등 (tailwind-all.scss, color.scss) 사용 시 컬러 지정 필요}

  • content : text

  • button :

    • href : 경로

    • target : value

    • style: btn-primary, btn-secondary, btn-text (버튼 가이드 참고)

    • size: btn-ultra-small, btn-small, btn-medium, btn-large (버튼 가이드 참고)

    • text : text

  • marginVertical : margin-top/bottom 간격에 대한 이중클래스 적용(spacing.scss, tailwind-all.scss 참고)
