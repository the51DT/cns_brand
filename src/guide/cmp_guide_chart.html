Chart

막대그래프 / 도넛그래프

  • 막대그래프 - type : (columnChart / invertChart / negativeChart)

  • 도넛그래프

  • 기업정보 > 재무정보, 주주현황/배당정보

  • 막대그래프 최대 5개 노출 (가운데 정렬)

  • 막대바 높이값 계산을 위한 임의 최대값(valueMax) 입력 필수

막대그래프(columnChart)

  • type : 'columnChart' - 그래프 타입

  • categories : X 값

  • series : 차트의 실제 데이터

  • categoryBarColor : 차트 컬러

  • valueMax : 최대값 *

막대그래프(invert)

  • type : invertChart - 음수가 있을 경우의 그래프

  • .chart-wrap__xAxis-area__line--y-intercept : Y좌표 0

  • series :

    • type : negative(음수) / positive(양수) // type 이 없을 경우 자동으로 양수 처리 적용

    • value : 값

막대그래프(negative)

  • type : negativeChart - 음수만 있을 경우의 그래프

  • .chart-wrap__xAxis-area__line--y-intercept : Y좌표 0

  • series :

    • type : negative(음수)

    • value : 값

    ]
