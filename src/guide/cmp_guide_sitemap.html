sitemap

  • menu1depthText : 1depth 텍스트 타입

  • menu1depth : 1depth 링크 타입

  • menu1depthLinkTarget : 1depth 링크 타입의 target 속성

  • menu2depth : 2depth 정의

  • menu2depth > categoryText : 2depth 대메뉴 텍스트 타입

  • menu2depth > categoryLink : 2depth 대메뉴 링크 타입

  • menu2depth > categoryLinkTarget : 2depth 대메뉴 target 속성

  • menu2depth > menu : 2depth 메뉴

  • menu2depth > menu > title : 2depth 메뉴 텍스트

  • menu2depth > menu > target : 2depth 메뉴 target 속성

  • menu2depth > menu > size : 2depth 메뉴 size (small / large)
