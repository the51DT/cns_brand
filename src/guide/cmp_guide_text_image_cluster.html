TextImageCluster: 범용적으로 사용하는 컴포넌트


옵션 설명

  • pcImg: 이미지를 삽입하는 경우 pcImg를 기본으로 사용

  • moImg: PC와 모바일 이미지가 다른 경우 moImg를 사용하면 tabletMini 분기점에서 자동으로 moImg로 전환

  • imgAlt: 이미지 대체 텍스트

  • textTitle: 큰 제목

  • descText: 설명 텍스트

  • button1Name: 첫 번째 버튼명
    button2Name: 두 번째 버튼명
    *각 버튼명 뒤에 Arrow를 포함하면 화살표 버튼으로 변경(ex. button1NameArrow)

  • textTop: 불리언 옵션, 기본값 false
    buttonFlex: 불리언 옵션, 기본값 false
    각각 텍스트가 위로 올지, 모바일 분기에서 버튼을 flex로 배치할지 결정

  • marginVertical: margin-top/bottom 간격에 대한 이중클래스 적용(spacing.scss, tailwind-all.scss 참고)
